A WordPress plugin to share posts on Twitter and Facebook like social media platform.

== Description ==
This simple plugin helps to share the single post to twitter and facebook like social media platform.

== Installation ==
1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==
= How do I use this plugin? =
 
Simple Post Share Buttons is automatically shows on single post pages. Just activate the plugin and see the magic.
 
= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
